Little Toy Kingdom


Description:


Little Toy Kingdom is a website that I have created using Adobe XD. It features a toy shop which sells various toys for kids, and a contact page in case customers want to get in touch with the store.

The store's main page features three sections, baby, boys, and girls toys. All of the features are also made with the prototype function in XD, which allows users to jump between each pages. The website also features a backstory, and a section for users to start a new case in case they need to. The footer was also added, so as to help visitors by adding information and navigation options at the bottom of web pages.
